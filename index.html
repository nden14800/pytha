三平方の定理

直角三角形が描く、面積の保存と宇宙の調和。
2500年の時を超え、人類の知性を支え続ける数学の至宝。
その証明、歴史、そして現代社会における驚くべき応用を徹底的に紐解く。

第1章：定理の本質と「面積の保存」

三平方の定理（Pythagorean Theorem）、欧米においてピタゴラスの定理として知られるこの命題は、ユークリッド幾何学において最も有名かつ、文明の発展に不可欠であった定理の一つである。数式として表現される $a^2 + b^2 = c^2$ は、中学生であれば誰もが暗記する公式だが、その真の美しさは単なる代数計算の便利さにあるのではない。その本質は、幾何学的な「面積の保存則」にある。

なぜ「長さ」を求める公式に「二乗」が登場するのだろうか？ 多くの学生が抱くこの疑問への答えは、古代ギリシャ人たちがこの定理をどう見ていたかを知ることで氷解する。彼らにとって、数とは「形」であった。$a^2$ とは、$a$ を二回掛ける計算ではなく、一辺が $a$ の「正方形そのもの」を意味していたのである。

ユークリッドの風車（The Windmill）

直角を挟む二辺 $a, b$ 上の正方形（青と赤）の面積の合計は、
斜辺 $c$ 上の正方形（紫）の面積と完全に一致する。

$$ a^2 + b^2 = c^2 $$

物理的な解釈：パズルのピース

上の図の中央にあるグレーの直角三角形に注目してほしい。この定理が主張しているのは、極めて物理的で触覚的な事実である。「二つの小さな正方形（青と赤の領域）をハサミで裁断してパズルのように組み合わせると、斜辺上の最も大きな正方形（紫の領域）を、一分の隙間もなく、かつ一片の余りもなく完全に埋め尽くすことができる」ということだ。

これは、液体を注ぎ替えることに例えることもできる。もし、青と赤の正方形が水槽で、深さが同じだと仮定しよう。その二つの水槽に入った水をすべて、紫の正方形の水槽に移し替えると、水面はまったく同じ高さになる。三平方の定理とは、宇宙における「量の保存」の一つの現れなのである。

豆知識：名前の由来と「勾股弦」

日本では「三平方の定理」と呼ばれるが、英語圏では「Pythagorean theorem（ピタゴラスの定理）」である。一方、中国などの漢字文化圏では古くから「勾股弦（こうこげん）の定理」と呼ばれてきた。直角三角形の短辺を「勾（かぎ）」、長辺を「股（もも）」、斜辺を「弦（つる）」に見立てた名称である。古代中国の数学書『周髀算経（しゅうひさんけい）』には、紀元前1000年頃にはすでにこの関係が知られていた記述がある。つまり、ピタゴラス一人の功績ではないのだ。

実践：定理を使った長さの求め方

このサイトの練習問題を解くために必要な、実践的な計算手順を解説する。公式 $a^2 + b^2 = c^2$ は変形することで、求めたい辺が「斜辺（一番長い辺）」か「それ以外の辺」かによって、2つの明確なアルゴリズムに分けることができる。

1 パターンA：斜辺 $c$ を求める（加算のアプローチ）

直角を挟む2辺（$a$ と $b$）が分かっていて、斜めの一番長い辺（$c$）を知りたいときは、面積を足し合わせるイメージを持つ。すなわち「2乗して、足して、ルートをとる」。

  1. 分かっている2つの辺を、それぞれ2乗する（$a^2$ と $b^2$）。
  2. それらを足し算する（$a^2 + b^2$）。これが斜辺の正方形の面積になる。
  3. その面積の平方根（ルート）をとることで、辺の長さに戻す。
$$ c = \sqrt{a^2 + b^2} $$

2 パターンB：斜辺以外の辺を求める（減算のアプローチ）

一番長い斜辺（$c$）と、もう一方の辺が分かっていて、残りの辺を知りたいときは、最大の面積から一部を切り取るイメージだ。すなわち「2乗して、引いて、ルートをとる」。必ず大きい方から小さい方を引くこと。

  1. 斜辺 $c$ と、もう一つの辺をそれぞれ2乗する。
  2. 斜辺の2乗から、もう一方の2乗を引き算する（$c^2 - b^2$）。
  3. その結果の平方根（ルート）をとる。
$$ a = \sqrt{c^2 - b^2} $$
豆知識：フェルマーの最終定理への架け橋

三平方の定理 $a^2 + b^2 = c^2$ において、自然数の組 $(a,b,c)$ は無数に存在する（例：3,4,5）。しかし、17世紀の数学者フェルマーは、この指数の「2」を「3以上」に変えたとき、つまり $a^n + b^n = c^n$ ($n \ge 3$) を満たす自然数の組は「一つも存在しない」と書き残した。これが360年間数学者を悩ませた「フェルマーの最終定理」である。ピタゴラスの定理は、数学史上最大の難問の出発点でもあったのだ。

第2章：歴史の霧と「ピタゴラス以前」の叡智

現代の教科書では「ピタゴラスの定理」として教えられるが、歴史的事実として、サモス島の賢人ピタゴラス（紀元前570年頃 - 紀元前495年頃）がこの定理の最初の発見者ではないことは、現代の歴史学・考古学においてほぼ確実視されている。ピタゴラス教団がこの定理の数秘術的な重要性を深く認識し、初めて厳密な証明を与えた可能性は高いが、人類によるその「利用」の歴史は、彼が生まれるよりもさらに千年以上前に遡ることができる。

バビロニアの粘土板：プリンプトン322

1920年代に発見され、コロンビア大学に所蔵されている紀元前1800年頃（ピタゴラスより1000年以上前）の古バビロニアの粘土板「プリンプトン322（Plimpton 322）」。一見するとただの商取引の記録に見えるが、解析の結果、そこには $(119, 120, 169)$ や $(3367, 3456, 4825)$ といった、巨大なピタゴラス数（三辺がすべて整数になる直角三角形の組み合わせ）のリストが60進法で整然と刻まれていることが判明した。これは、当時のバビロニア人が高度な数論的知識を持っていた決定的な証拠である。

古代エジプト：縄張り師（ハルペドナプタエ）

ナイル川は毎年氾濫し、肥沃な土壌をもたらすと同時に、農民たちの土地の境界線をすべて泥の下に消し去った。洪水のたびに境界を正確に引き直す必要があった古代エジプトでは、ファラオに仕える測量技師たちが活躍した。 彼らは「縄張り師（ハルペドナプタエ）」と呼ばれ、等間隔に12個の結び目を作った長いロープを持っていた。このロープで $3:4:5$ の三角形を作ることで、地面に完璧な「直角」を出現させたのである。ピラミッドの底面が驚くほど正確な正方形であるのも、この技術のおかげかもしれない。

古代インド：シュルバ・スートラ

インドにおいても、紀元前800年頃には祭壇を作成するための規定書『シュルバ・スートラ』の中で、直角三角形の性質が記述されていた。彼らは「長方形の対角線は、その両辺（縦と横）が個別に作る領域（正方形）を合わせたものと同じ領域を作る」と表現しており、儀式における祭壇の面積を正確に制御するためにこの定理を用いていた。

豆知識：無理数の発見と死

ピタゴラス教団は「万物は数（整数または整数の比）である」という教義を信じていた。しかし、皮肉なことに彼ら自身が崇める三平方の定理を使えば、一辺が1の正方形の対角線は $\sqrt{2}$ となり、整数比で表せない「無理数」が出現してしまう。伝説によれば、教団の一員ヒッパソスがこの不都合な真実を外部に漏らしたため、教団の怒りを買い、船から海に突き落とされて殺されたと言われている。真理の探究には命がけの代償が必要だったのだ。

第3章：証明の万華鏡

三平方の定理ほど、多様な方法で証明された定理は他に類を見ない。現在までに確認されている証明法は数百種類に及び、万能の天才レオナルド・ダ・ヴィンチや、第20代アメリカ大統領ジェームズ・ガーフィールド、さらには物理学者アインシュタインまでもが独自の証明を試みている。ここでは代表的なアプローチを紹介する。

1. 正方形分割法（パズル的アプローチ）

最も直感的で視覚的にわかりやすいのは、一辺が $a+b$ の大きな正方形を二通りに分割する方法である。 一つは四隅に４つの直角三角形を配置し、中央に $c^2$ の正方形を作る方法。 もう一つは、長方形を作るように三角形を配置し、$a^2$ と $b^2$ の正方形を作る方法。 外枠の正方形の面積は同じであり、中の4つの三角形の面積も同じである。したがって、残りの余白部分の面積は必然的に等しくならなければならない。すなわち、$c^2 = a^2 + b^2$ である。

2. ガーフィールド大統領の台形による証明

1876年、当時下院議員だったジェームズ・ガーフィールド（後の第20代大統領）は、二つの合同な直角三角形を並べて「台形」を作り、その面積計算から定理を導いた。 台形の面積を「(上底+下底)×高さ÷2」で計算したものと、構成要素である「3つの三角形の面積の和」として計算したものを等式で結ぶと、見事に $a^2 + b^2 = c^2$ が現れる。政治家が数学のオリジナル証明を残すことは極めて稀である。

3. アインシュタインの相似による証明

若き日のアインシュタインが見出したとされる証明は、面倒な面積計算を行わずに「形（相似）」に注目する極めてエレガントなものである。 直角三角形の直角の頂点から斜辺に垂線を下ろすと、元の三角形と相似な二つの小さな三角形が現れる。「面積比は辺の二乗に比例する」という相似の性質を使えば、$c^2 = a^2 + b^2$ という関係式が一瞬で、魔法のように導かれる。

豆知識：ピタゴラス数の不思議な性質

$(3, 4, 5)$ や $(5, 12, 13)$ のような、三辺が整数のピタゴラス数には、面白い法則がある。
1. 少なくとも一つは必ず3の倍数である。
2. 少なくとも一つは必ず4の倍数である。
3. 少なくとも一つは必ず5の倍数である。
試しに $(5, 12, 13)$ を見ると、12は3と4の倍数、5は5の倍数となっており、全て満たしている。

第4章：現代社会を動かす隠れたエンジン

古代の遺跡から発掘された定理は、現代において廃れるどころか、デジタル社会の基盤として毎秒数兆回も計算されている。もし明日、三平方の定理が使えなくなれば、私たちの文明は即座に崩壊するだろう。

GPSと三次元測位

私たちがスマートフォンで現在地を知ることができるのは、GPS衛星のおかげである。カーナビやGoogleマップは、上空2万kmにある複数のGPS衛星からの電波の到達時間差を測定し、三平方の定理を三次元空間へ拡張した距離公式 $$ d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2} $$ を用いて位置を特定している。衛星の位置とあなたの位置を結ぶ巨大な直角三角形が、宇宙空間に描かれているのだ。

CG技術とゲーム・メタバース

最新の3Dビデオゲームや映画のVFX、そしてメタバースにおいて、キャラクターの動き、光の反射、影の描写は、すべて空間内のベクトル計算に基づいている。 例えば、FPSゲームで「敵が射程圏内にいるか」を判定する際、プログラムは敵の座標とプレイヤーの座標から三平方の定理を使って距離を瞬時に計算している。画面上のあらゆるリアリティは、この定理の高速演算によって支えられている。

建築とA判用紙

大工は今でも「サシゴ（3:4:5）」を使って家の土台の直角を確認する。また、私たちが日常的に使うコピー用紙「A4」や「B5」の縦横比 $1:\sqrt{2}$ も、直角二等辺三角形から導かれる比率である。この比率のおかげで、紙を半分に折っても元の形と相似になり、資源を無駄にしない効率的な印刷が可能になっている。

豆知識：野球と盗塁

野球のダイヤモンド（内野）は一辺が90フィート（約27.43m）の正方形である。では、キャッチャーが2塁へ盗塁を阻止するためにボールを投げるとき、その距離はどれくらいだろうか？ 三平方の定理を使えば $\sqrt{90^2 + 90^2} \approx 127.3$フィート（約38.8m）とすぐに計算できる。選手たちは無意識のうちに、この距離感を体に染み込ませてプレーしているのだ。

永遠の等式

文明が滅び、言語が失われたとしても、
直角三角形の辺上に描かれた三つの正方形の面積の関係を見れば、
知的な生命体はそれが真理であることを理解するだろう。
三平方の定理は、人類が発明したものではなく、
宇宙開闢の瞬間からそこに存在していた「構造」を発見したものに過ぎない。
それは、私たちが星空を見上げるとき、あるいはスマートフォンの画面を見るとき、
常に静かに、しかし力強く世界を支え続けている。
